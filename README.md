# 📞 CallBeacon Web

A modern and responsive landing page for **CallBeacon**, an Android application that automatically sends personalized SMS replies for missed calls. The website showcases the app's features, provides installation instructions, and allows users to download the APK.

## 🌟 Overview

CallBeacon helps users stay connected even when they can't answer their phone. Whether you're driving, in a meeting, or your phone is on silent, CallBeacon automatically sends a custom SMS to missed callers, letting them know why you're unavailable.

This repository contains the **official promotional website** for the application—not the Android app itself.

## 📱 About CallBeacon

CallBeacon automatically detects missed calls and sends personalized SMS replies based on your availability.

Examples include:
- 🚗 Driving
- 🤫 Silent Mode
- 💼 In a Meeting
- 📚 Studying
- 😴 Sleeping
- ✈️ Traveling

No internet connection is required—the SMS is sent directly using your mobile network.
